How to Use

  • Click any element to see detailed properties and 3D Bohr model
  • Temperature toggle: slide to see phase states (solid/liquid/gas) at any temperature
  • Trends toggle: color-code by electronegativity, atomic radius, ionization energy
  • Year toggle: see when each element was discovered
  • Search: type name, symbol, or atomic number to highlight elements
  • Explorer: visualize 3D molecular structures
  • Calculator: compute molar masses from formulas
